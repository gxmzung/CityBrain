# Interview Notes Evidence

## Purpose

이 폴더는 CityBrain의 문제 검증을 위한 직원/학생 인터뷰 기록을 정리하는 공간이다.

Day 2 작업에서 실제 인터뷰를 진행한 뒤 이 폴더에 기록한다.

---

## Staff Interview Questions

```text
1. 식수나 판매량은 현재 어떤 방식으로 기록하시나요?
2. 메뉴별 선호도나 학생 반응은 따로 확인하시나요?
3. 인기 메뉴가 빨리 소진되는 경우가 있나요?
4. 학생들이 메뉴나 혼잡도 관련해서 자주 묻는 편인가요?
5. 앱으로 남은 수량이나 혼잡도를 미리 볼 수 있다면 운영에 도움이 될까요?
6. 식자재 주문이나 메뉴 수량을 정할 때 참고하는 데이터가 있나요?
7. 운영자 입장에서 입력이 부담되지 않으려면 어떤 방식이 가장 좋을까요?
```

---

## Student Interview Questions

```text
1. 학식 먹으러 갔다가 줄이 길어서 포기한 적이 있나요?
2. 먹고 싶은 메뉴가 소진돼서 아쉬웠던 적이 있나요?
3. 오늘 메뉴를 어디서 확인하나요?
4. 혼잡도나 남은 수량을 앱에서 볼 수 있으면 사용할 것 같나요?
5. 메뉴 만족도나 후기 기능이 있으면 참고할 것 같나요?
6. 학식 앱에서 가장 필요한 기능 하나만 고르면 무엇인가요?
```
