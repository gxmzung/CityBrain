# Production Gap

CityBrain is currently an MVP-level smart campus service.

Before production deployment, the following areas must be improved:

## Authentication

- Student login
- Admin login security
- Role-based permissions

## Privacy

- Privacy policy
- Data retention policy
- Consent flow

## Reliability

- Error monitoring
- Backup strategy
- Admin recovery flow

## Deployment

- HTTPS
- Domain setup
- Server monitoring
- Store release preparation
