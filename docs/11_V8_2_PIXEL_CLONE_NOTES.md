# CityBrain V8.2 Pixel Clone Notes

V8.2는 기존 V8/V8.1의 재해석형 UI가 아니라, 제공된 1~5번 시안의 화면 구조를 그대로 따라가도록 Android Compose 화면을 재작성한 버전입니다.

## 적용 원칙
- 큰 파란 헤더 제거
- 흰 배경 + 작은 상단 타이틀 구조
- 홈 / 실시간 / 메뉴 / 공지 / 참여 화면을 시안 순서대로 재구현
- 단일 학생식당 구조 유지
- 주변 선택지 제거
- MY 탭 제거
- 버튼/칩/별점/제보 항목은 실제 선택 상태가 바뀌도록 구현

## 주의
- 외부 이미지 asset 없이 구현하므로 음식 사진은 emoji/색상 블록으로 대체했습니다.
- Android 기기별 폰트/상태바/네비게이션바 차이 때문에 시안과 100% 픽셀 동일하지는 않을 수 있습니다.
